REQUIRES TEMBOO LIBRARY

To download:
1. signup for free account @ http://temboo.com
2. download and make sure to include the referenced library files:
  temboo.flickr.php
  temboo.google.php
  temboo.instagram.php
  temboo.twitter.php
3. rename config.php.template to config.php
4. update config.php to contain your own configuration values

PHP version 5
